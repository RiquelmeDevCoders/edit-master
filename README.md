# 🎬 Edit Master - Landing Page de Conteúdo Estratégico

Uma landing page imersiva, moderna e de alta performance desenvolvida para a **Edit Master**, uma agência especializada em edição de vídeos e estratégia de conteúdo para o mercado digital.

🔗 **Acesse o projeto online:** [edit-master.netlify.app](https://edit-master.netlify.app)

## 🚀 Sobre o Projeto

Este projeto é uma **Single Page Application (SPA)** construída com foco em experiência do usuário (UX), animações fluidas e design responsivo. O objetivo é transmitir autoridade e modernidade através de uma estética "Dark/Space" com elementos em neon e vidro (Glassmorphism).

O site apresenta o portfólio da agência, prova social através de áudios reais e explicações detalhadas sobre os serviços prestados.

## ✨ Funcionalidades Principais

* **Showcase de Vídeos (Portfólio):** Sistema de abas interativas para filtrar projetos por categoria (SaaS, Gateway, Infoprodutos, etc.).
* **Player de Áudio Customizado:** Uma implementação única de players de áudio para a seção de feedbacks, com visualização de ondas sonoras animadas e controle de Play/Pause.
* **Animações de Alta Performance:** Uso intensivo de *Framer Motion* para transições de entrada, hover effects e interações suaves.
* **Background "StarField":** Um fundo animado com partículas (estrelas) renderizado via HTML5 Canvas para profundidade visual.
* **Design Responsivo:** Layout totalmente adaptável para dispositivos móveis, tablets e desktops.
* **FAQ Interativo:** Seção de perguntas frequentes com acordeões animados.

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **[React.js](https://react.dev/):** Biblioteca principal para construção da interface.
* **[Tailwind CSS](https://tailwindcss.com/):** Framework de estilização para design rápido e consistente.
* **[Framer Motion](https://www.framer.com/motion/):** Biblioteca para animações complexas e gestos.
* **[Lucide React](https://lucide.dev/):** Conjunto de ícones leves e customizáveis.
* **[Clsx & Tailwind-Merge](https://www.npmjs.com/package/clsx):** Utilitários para renderização condicional de classes CSS.

## 📂 Estrutura do Projeto

* `AudioPlayer`: Componente lógico e visual para reprodução de feedbacks de clientes.
* `MagicButton`: Botão com efeitos de borda animada e brilho (glow).
* `StarField`: Componente canvas para o background estelar.
* `MotionShowcase`: Galeria de vídeos com filtros de categoria.

## 📦 Como rodar localmente

1.  Clone este repositório:
    ```bash
    git clone [https://github.com/RiquelmeDevCoders/edit-master.git](https://github.com/seu-usuario/edit-master.git)
    ```

2.  Instale as dependências:
    ```bash
    cd edit-master
    npm install
    ```

3.  Execute o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
